## 이미지 프롬프트 생성 요청

위 심리학 주제에 적합한 유튜브 썸네일 및 영상 삽입 이미지를 생성하기 위한 나노바나나프로 프롬프트를 만들어주세요.

### 이미지 유형

#### 1. 썸네일 이미지
- **용도**: 유튜브 썸네일
- **크기**: 16:9 비율
- **스타일**: 시선을 사로잡는 비주얼
- **구성 요소**:
  - 주제를 상징하는 중심 이미지
  - 감정을 불러일으키는 색상 팔레트
  - 텍스트 삽입 공간 확보 (주로 상단 또는 하단)

#### 2. 본문 삽입 이미지 (3-5개)
- **용도**: 영상 중간 설명용
- **크기**: 16:9 비율
- **스타일**: 깔끔하고 전문적
- **구성 요소**:
  - 각 섹션 주제를 시각화
  - 일관된 색상 톤
  - 심플하고 이해하기 쉬운 비주얼

### 나노바나나프로 프롬프트 형식

```
[썸네일]
프롬프트: {{상세한 이미지 설명}}
스타일: {{원하는 스타일 - 예: 미니멀리즘, 일러스트, 사진 같은}}
색상: {{메인 색상 팔레트}}
분위기: {{감정/무드 - 예: 차분한, 역동적인, 따뜻한}}

[본문 이미지 1 - {{주제}}]
프롬프트: {{상세한 이미지 설명}}
스타일: {{원하는 스타일}}
색상: {{메인 색상 팔레트}}

[본문 이미지 2 - {{주제}}]
프롬프트: {{상세한 이미지 설명}}
스타일: {{원하는 스타일}}
색상: {{메인 색상 팔레트}}

...
```

### 스타일 가이드
- 심리학 주제에 어울리는 전문적이면서도 친근한 느낌
- 과도한 클리셰 피하기 (예: 뇌 이미지 남발 지양)
- 한국 시청자에게 친숙한 시각적 요소
- 밝고 긍정적인 색상 팔레트 선호
