## 생성 요청

위 트렌딩 주제를 바탕으로 10-15분 분량의 유튜브 롱폼 스크립트를 작성해주세요.

### 스크립트 구성 요소

1. **훅/도입부 (30초-1분)**
   - 시청자의 주의를 끄는 강력한 질문이나 사실
   - 이 영상을 봐야 하는 이유 제시

2. **본론 (8-12분)**
   - 주제에 대한 심층적인 설명
   - 실생활 예시 3-5개 포함
   - 과학적 연구 또는 전문가 의견 인용
   - 단계별 설명 또는 체계적 구조

3. **실천 방법 (2-3분)**
   - 시청자가 바로 적용할 수 있는 구체적인 팁
   - 단계별 가이드라인

4. **마무리 (30초-1분)**
   - 핵심 내용 요약
   - 행동 촉구 (구독, 좋아요, 댓글)
   - 다음 영상 예고

### 스타일 가이드
- 친근하고 대화하는 듯한 톤
- 전문 용어는 쉽게 풀어서 설명
- 시청자에게 직접 말하는 2인칭 사용
- 적절한 감정 표현과 공감
- 문단 사이사이 자연스러운 전환

### 추가 요구사항
- 영상 길이: 10-15분 (약 2,500-3,500 단어)
- 한국어로 작성
- 자막 편집을 고려한 적절한 문장 길이
- 중요한 포인트는 **강조** 표시
