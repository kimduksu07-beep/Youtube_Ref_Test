# 유튜브 롱폼 자동화 프로그램

## 프로젝트 목적
심리학 유튜브 트렌딩 주제 수집 → 스크립트 생성용 프롬프트 자동 생성 → 나노바나나프로 이미지 프롬프트 자동 생성

## 사용자 수준
코딩 경험 전무한 초보자. 모든 안내를 한국어로, 모든 에러를 이해 가능한 수준으로 표시할 것.

## 기술 스택
- TypeScript, Node.js
- YouTube Data API v3
- inquirer (CLI 선택 UI)
- clipboardy (클립보드 복사)
- dayjs (날짜 처리)
- dotenv (환경변수)
- chalk (터미널 색상)

## 실행 방법
- 설치: npm install
- 실행: npm start

## 핵심 규칙
- API 키는 반드시 .env에서 관리
- 모든 터미널 출력은 한국어
- 모든 커밋 메시지와 Git 관련 작업은 한국어로 작성
- 에러 발생 시 "무엇이 잘못되었고, 어떻게 해결하는지" 한국어로 안내
- 결과물은 output/{날짜}_{주제}/ 폴더에 저장
- 파일 인코딩은 UTF-8
- 이미지 프롬프트의 캐릭터는 반드시 채널 공식 캐릭터(둥근 흰색 2등신, 동그란 안경, 남색 후디, PSYCH 노트)를 사용한다

## 코딩 규칙
- async/await 사용
- 모든 함수에 JSDoc 한국어 주석
- try-catch로 에러 처리
- console.log에 이모지 활용하여 상태 직관적 표시
