# 🎬 유튜브 롱폼 자동화 프로그램

심리학 유튜브 트렌딩 주제를 자동으로 수집하고, 스크립트 및 이미지 프롬프트를 생성하는 자동화 도구입니다.

## 📋 프로젝트 소개

이 프로그램은 다음과 같은 작업을 자동화합니다:

1. **트렌딩 주제 수집**: YouTube Data API를 통해 심리학 관련 인기 영상을 검색
2. **주제 선택**: 터미널에서 원하는 주제를 선택
3. **스크립트 프롬프트 생성**: 선택한 주제로 Claude.ai에 붙여넣을 스크립트 생성 프롬프트 자동 생성
4. **이미지 프롬프트 생성**: 나노바나나프로에 사용할 이미지 프롬프트 자동 생성
5. **클립보드 복사**: 생성된 프롬프트를 자동으로 클립보드에 복사

## 🚀 설치 방법

### 1. Node.js 설치 확인

터미널(명령 프롬프트)을 열고 다음 명령어를 입력하세요:

```bash
node --version
```

Node.js가 설치되어 있지 않다면 [Node.js 공식 사이트](https://nodejs.org/)에서 다운로드하여 설치하세요.

### 2. 프로젝트 의존성 설치

프로젝트 폴더에서 다음 명령어를 실행하세요:

```bash
npm install
```

모든 필요한 패키지가 자동으로 설치됩니다.

## 🔑 API 키 설정

### YouTube Data API 키 발급

1. [Google Cloud Console](https://console.cloud.google.com/)에 접속
2. 새 프로젝트 생성 또는 기존 프로젝트 선택
3. "API 및 서비스" > "라이브러리"로 이동
4. "YouTube Data API v3" 검색 후 활성화
5. "사용자 인증 정보" > "사용자 인증 정보 만들기" > "API 키" 선택
6. 생성된 API 키 복사

### .env 파일 설정

프로젝트 폴더의 `.env` 파일을 열고 다음과 같이 API 키를 입력하세요:

```
YOUTUBE_API_KEY=여기에_발급받은_API_키를_붙여넣으세요
```

## 💻 사용 방법

### 프로그램 실행

터미널에서 다음 명령어를 입력하세요:

```bash
npm start
```

### 실행 흐름

1. **트렌딩 주제 수집**: 프로그램이 자동으로 심리학 관련 인기 영상을 검색합니다
2. **주제 선택**: 화면에 표시된 주제 목록에서 원하는 주제를 선택합니다
3. **프롬프트 생성**:
   - 스크립트 생성용 프롬프트가 자동으로 생성되고 클립보드에 복사됩니다
   - 이미지 생성용 프롬프트가 자동으로 생성되고 클립보드에 복사됩니다
4. **결과 저장**: `output/` 폴더에 모든 결과물이 저장됩니다

### Claude.ai에서 스크립트 생성

1. [Claude.ai](https://claude.ai)에 접속
2. 클립보드에 복사된 스크립트 프롬프트를 붙여넣기 (Ctrl+V)
3. 생성된 스크립트를 영상 제작에 활용

### 나노바나나프로에서 이미지 생성

1. 나노바나나프로에 접속
2. 클립보드에 복사된 이미지 프롬프트를 붙여넣기
3. 생성된 이미지를 영상에 활용

## 📁 프로젝트 구조

```
youtube-automation/
├── src/                    # 소스 코드
│   ├── index.ts           # 메인 진입점
│   ├── youtube/           # YouTube API 관련
│   ├── prompt/            # 프롬프트 생성
│   ├── ui/                # 터미널 UI
│   └── utils/             # 유틸리티 함수
├── templates/             # 프롬프트 템플릿
├── output/                # 결과물 저장
├── .env                   # 환경 변수 (API 키)
└── package.json           # 프로젝트 설정
```

## 🛠️ 기술 스택

- **TypeScript**: 타입 안전성을 갖춘 JavaScript
- **Node.js**: JavaScript 런타임
- **YouTube Data API v3**: 유튜브 데이터 수집
- **inquirer**: 터미널 선택 UI
- **clipboardy**: 클립보드 자동 복사
- **dayjs**: 날짜 처리
- **dotenv**: 환경 변수 관리
- **chalk**: 터미널 색상 출력

## ⚠️ 문제 해결

### "YouTube API 키가 설정되지 않았습니다" 오류

`.env` 파일에 `YOUTUBE_API_KEY`가 제대로 입력되었는지 확인하세요.

### "npm을 찾을 수 없습니다" 오류

Node.js가 제대로 설치되었는지 확인하고, 터미널을 재시작해보세요.

### API 할당량 초과 오류

YouTube Data API는 일일 할당량이 있습니다. 다음 날 다시 시도하거나, Google Cloud Console에서 할당량을 늘려야 합니다.
